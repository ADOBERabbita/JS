- Là một tác động nào đó lên các đối tượng HTML. Qua đó ta có thể bắt đc sự kiện và viết một đoạn JS để thực thi một chương trình nào đó. - Cú pháp: element.eventname = funcion() { // Code } - Tong đó: + element: là phần tử muốn bắt sự kiện + eventname: tên sự kiện - Một số DOM Events phổ biến: + onload : khi trình duyệt đã load xong mọi thứ thì những đoạn code nằm bên trong đó mưới đc chạy + onblur : kích hoạt khi 1 phần tử mất trọng tâm (ko đc focus vào nữa) + onfocus : kích hoạt khi 1 phần tử đúng tọng tâm (đc focus vào) + onkeydown: kích hoạt khi 1 phím đc nhấn + onkeyup : kích hoạt khi 1 phím đc nhả ra + onclick : kích hoạt trên con chuột vừa nhấn vào phần tử + onchange : kích hoạt khi gtri đc thay đổi khác đi so với gtri trước đó
