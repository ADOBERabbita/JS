1. Strict mode là gì? - Strict mode là chế độ code nghiêm ngặt, nó bắt buộc lập trình viên phải tuân thủ theo quy tắc mà javascript đưa ra. Các lỗi cơ bản thường gặp như khai báo trùng tên biến và trùng tên hàm, sử dụng biến mà chưa được định nghĩa. - Để sử dụng strict mode ta khai báo "use strict"; ở trước vị trí mà ta muốn sử dụng. 2. Có 2 kiểu Strict Mode - Strict mode toàn cục: Khai báo bên ngoài cùng. - Strict mode cục bộ: Khai báo bên trong một hàm.
